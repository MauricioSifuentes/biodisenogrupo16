¿Qué es Mobirise?

Mobirise es una app offline para Windows y Mac para crear fácilmente sitios web pequeños o medianos, páginas de inicio, resúmenes online y portafolios, sitios promocionales para apps, eventos, servicios y productos. 

¿Por qué Mobirise?

* Minimalista y con una interface de usuario extremadamente fácil de usar.
* Diseños para dispositivos móviles y con los últimos bloques y técnicas para crear sitios web.
* Gratis para uso comercial y sin fines de lucro.
